# 비개발자용 – HubSpot에 붙여넣기 안내

개발이 아니어도 아래만 따라하시면 됩니다.

---

## 할 일은 두 가지뿐입니다

1. **헤드·바디가 이미 들어 있는 파일**을 연다.
2. **그 내용 전체**를 HubSpot `solution-demo.html`에 **통째로** 붙여넣는다.

---

## 방법 1: 데모 페이지 전체를 쓰고 싶을 때 (추천)

**쓰는 파일:** `hubspot/templates/demo-page.html`

- 이 파일 안에는 **head**, **body**, **필수 태그**(standard_header_includes, standard_footer_includes)가 **이미 다 들어 있습니다.**
- 별도로 head/body를 구성할 필요 없습니다.

**순서:**

1. 로컬에서 **`hubspot/templates/demo-page.html`** 파일을 연다.
2. **Ctrl+A** (전체 선택) → **Ctrl+C** (복사).
3. HubSpot Design Manager에서 **solution-demo.html**을 연다.
4. 편집기 안에서 **Ctrl+A** (전체 선택) → **Ctrl+V** (붙여넣기).
5. **file_base** 한 줄만 수정한다.  
   - 상단에 `{% set file_base = '...' %}` 가 있으면,  
   - `'...'` 안에 **파일 매니저에서 style.css URL 복사 후, 맨 뒤 `/css/style.css` 를 뺀 주소**를 넣는다.  
   - 예: `https://22319584.fs1.hubspotusercontent-na1.net/hubfs/22319584/solution-demo`
6. **Publish changes** 로 저장.

이렇게 하면 **head/body 구성 + 필수 태그**는 이미 되어 있고, 에러 1~4도 없어집니다.

---

## 방법 2: 일단 에러만 없애고 싶을 때 (최소 템플릿)

**쓰는 파일:** `hubspot/HubSpot-최소-템플릿.html`

- **head**와 **body**가 이미 구성되어 있고,  
  `{{ standard_header_includes }}` 와 `{{ standard_footer_includes }}` 도 **이미 들어 있습니다.**
- 화면에는 “본문 자리입니다” 같은 안내 문구만 보입니다.

**순서:**

1. 로컬에서 **`hubspot/HubSpot-최소-템플릿.html`** 파일을 연다.
2. **Ctrl+A** (전체 선택) → **Ctrl+C** (복사).
3. HubSpot Design Manager에서 **solution-demo.html**을 연다.
4. 편집기 안에서 **Ctrl+A** (전체 선택) → **Ctrl+V** (붙여넣기).
5. **file_base** 한 줄만 수정한다.  
   - `{% set file_base = '여기에_...' %}` 에서  
   - `'여기에_...'` 를 **파일 매니저에서 style.css URL 복사 후, 맨 뒤 `/css/style.css` 를 뺀 주소**로 바꾼다.
6. **Publish changes** 로 저장.

이렇게 하면 **head/body 구성 + 필수 태그**가 되어 있어서, 에러 1~4가 사라집니다.  
(나중에 데모 전체를 쓰려면 방법 1의 `demo-page.html` 전체로 다시 붙여넣으면 됩니다.)

---

## 정리

| 하고 싶은 것 | 사용할 파일 | head/body |
|-------------|-------------|-----------|
| 데모 페이지 전체 사용 | `hubspot/templates/demo-page.html` | 이미 구성됨 |
| 일단 에러만 없애기 | `hubspot/HubSpot-최소-템플릿.html` | 이미 구성됨 |

**공통:**  
- head, body는 **이미 파일에 구성되어 있음.**  
- **필수 태그**도 **이미 들어 있음.**  
- 할 일은 **파일 열기 → 전체 복사 → HubSpot solution-demo.html에 붙여넣기 → file_base만 수정 → 저장**입니다.
