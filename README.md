# 이벤터스 솔루션 데모 페이지

플랫폼 연동 없이 독립적으로 작동하는 정적 웹사이트입니다.

## 파일 구조

```
solution demo/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css      # 스타일시트
├── js/
│   ├── data.js        # 로컬 데이터 관리
│   └── main.js        # 메인 로직
└── README.md          # 프로젝트 설명
```

## 주요 기능

1. **전화번호 입력**: 01041283217 입력 시 바로 상세페이지로 이동
2. **참가자 정보 입력**: 이름, 이메일, 전화번호, 비밀번호, 약관동의
3. **솔루션 체험 기능**:
   - 공지사항
   - 행사 정보
   - 연사 프로필
   - 파일 다운로드
   - Q&A
   - 투표
   - 경품 추첨
   - 경연대회
   - 웨비나 시청
   - 출석 체크

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 전화번호 입력 화면에서 `01041283217` 입력
3. 참가자 정보 입력 후 데모 화면에서 기능 체험

## 데이터 저장

모든 데이터는 브라우저의 로컬스토리지에 저장됩니다.
페이지를 새로고침해도 데이터가 유지됩니다.

## 브라우저 호환성

- 모바일 브라우저 (iOS Safari, Chrome Mobile)
- 데스크톱 브라우저 (Chrome, Firefox, Safari, Edge)
